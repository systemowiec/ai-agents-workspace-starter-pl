# API Map - All Endpoints

> Źródło prawdy dla mapy endpointów. Aktualizuj po każdym dodaniu/zmianie endpointu.

## Conventions

- Base URL: `/api/v1/`
- URL: kebab-case
- Body: snake_case
- Response envelope: `{ "data": ..., "meta": ..., "errors": [...] }`
