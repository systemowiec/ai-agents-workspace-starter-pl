# Agent Learnings

Ten katalog zawiera wiedzę zdobytą przez agentów podczas pracy nad projektem.

## Pliki

- `gotchas.md` - pułapki, nietypowe zachowania, rzeczy łatwe do przeoczenia
- (dodawaj kolejne pliki w miarę jak projekt rośnie)

## Format wpisu

```markdown
### [krótki tytuł]
**Kontekst**: [gdzie/kiedy to występuje]
**Problem**: [co się stało]
**Rozwiązanie**: [jak to naprawić]
**Data**: [YYYY-MM-DD]
```

## Zasady

- Agent NIE dodaje wpisów automatycznie - zawsze pyta użytkownika
- Wpisy są wersjonowane w Git jak każdy inny kod
- Code review na wpisy jest opcjonalny, ale zalecany
- Usuwanie wpisów wymaga potwierdzenia użytkownika
